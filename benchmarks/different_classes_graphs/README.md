## Файлы бенчмарков для логического синтеза

    external.jsonl - 201 схема из ранее опубликованных бенчмарков. Схемы адаптированны для проекта
    random.jsonl - 125 случайных схем, сгенерированных скриптом
    standard.jsonl - 20 небольших типовых схем
    const.jsonl - схемы с константами
    random_size.jsonl - случайные схемы разного размера
    random_size_xor.jsonl - случайные схемы разного размера c операцией xor
    small.jsonl - схемы небольшого размера, для проверки различных случаев
    standard_blocks.jsonl - схемы для типовых блоков разного размера
